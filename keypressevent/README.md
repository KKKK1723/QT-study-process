# QtCreator 按键事件 Demo

## 项目概述
这个 Demo 是在 **QtCreator** 上开发的，使用了 `KeyPressEvent` 按键事件。我们实现了一个简单的界面功能，具体需求如下：
- 按住 `Ctrl + M` 将界面切换为全屏模式。
- 按下 `Esc` 键将界面从全屏模式缩小恢复。
- 按下 `上、下、左、右` 方向键时，可以移动界面上的按键。

## 实现功能

### 1. 全屏与缩小功能
- **全屏模式**  
  - 触发条件：按住 `Ctrl` 并同时按下 `M` 键。  
  - 效果：界面切换为全屏显示。
- **缩小恢复**  
  - 触发条件：按下 `Esc` 键。  
  - 效果：界面从全屏模式恢复到原始大小。

### 2. 按键移动功能
- **方向键控制**  
  - 按下 `上` 键：按键向上移动。  
  - 按下 `下` 键：按键向下移动。  
  - 按下 `左` 键：按键向左移动。  
  - 按下 `右` 键：按键向右移动。

## 运行说明
1. 在 QtCreator 中打开项目。
2. 编译并运行程序。
3. 操作界面时：
   - 使用 `Ctrl + M` 进入全屏。
   - 使用 `Esc` 退出全屏。
   - 使用方向键（`↑` `↓` `←` `→`）移动按键。
